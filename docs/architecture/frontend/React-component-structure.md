#### Types

For a local child Component that uses mostly its parent props - use `react`'s `ComponentProps<typeof Comp>`.

To make a Component customizable with chakra styles - use chakra props as eg `ComponentProps<typeof Box>`.

Type React Router page Component with its `Route.ComponentProps` from `import type { Route } from "~/react-router/posts/detail/+types/index"`, eg as `props.params.id`. Those types are generated by `mise lint`.

We must keep `props` types inlined - no single-use `interface Props`.
