The <Term id="backend" /> runs a <LinkExt href="https://docs.djangoproject.com/en/5.2/ref/contrib/admin/">Django Admin</LinkExt> panel
at <LinkExt href={`${env.VITE_SERVER_URL}/admin/`}>{env.VITE_SERVER_URL}/admin/</LinkExt>.

Only <PageLink id="User" hash="links" />s with the flags `is_staff = true` or `is_superuser = true` can access it.

It's mostly generated based on the source code defining the SQL Database Tables, but I added a number of admin-level actions.
