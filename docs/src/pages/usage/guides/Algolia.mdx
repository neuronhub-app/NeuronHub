Algolia is used on all list pages in the <Term id="frontend" />, with data indexed by the <Term id="backend" />.

Indexes: `posts`, `jobs`, `profiles` - each suffixed with the environment name (eg `jobs_stage`).


## Rules

Rules follow `if -> then` - affecting both hits and filters (facets).

They can be setup in the <LinkExt href="https://dashboard.algolia.com/apps/OJJMNNBR0H/rules/jobs_dev_viktor">Dashboard</LinkExt> using the visual editor.

And then toggled by the UI, users, or anywhere using the URL params as `?enableRules=false`.
