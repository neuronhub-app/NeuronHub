This primitive CMS allows to customize a few elements.

In the admin you can re-order all sections/links by drag-and-drop, and then pressing the `Save` button.


<Notice>
  Django Admin has poor UI, as it's auto-generated.<br />
  Yet it's easy to extend, and has no maintenance cost.
</Notice>
