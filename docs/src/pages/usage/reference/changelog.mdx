<ChangelogEntry title="rc.9" date="2026-04-02" version="v0.3.2-rc.9">

  <ChangelogTag tag="New" />
  - A floating button for opening Django Admin or documentation:
    <ImageWithDialog size="xs" src="https://media.neuronhub.app/docs/changelog/admin-only-nav-button.avif"/>


  <ChangelogTag tag="Improvements" /> __Django Admin__
  - Customize `Reply-To: {contact_email}` emails header in <PageLink id="SiteConfig"/>,
    instead of defaulting to `SiteConfig.sender_email`.
  - Add buttons to reset timezone on a <AdminLink id="JobAlert" />, and set it to PT
    - Useful for triggering alerts from admin, as otherwise it only triggers `JobAlert`s with local time matching 8am-9am.
  - Enable Markdown editor for `Job.description` field.
  - Enable `History` on <AdminLink id="SiteConfig" />.
  - Richer <PageLink id="Sentry"/> logs during `JobAlert`s sending.

  <ChangelogTag tag="UI" />
  - Fix `/subscriptions` page on mobile.
  - See Sofi's reports.

  <ChangelogTag tag="Fixes" />
  - In emails sent from the `Contact` modal - set `Reply-To` to user's email (if any),
    instead of appending `Email from: {email}` to the email body.
  - Customize emails header `From: {sender_email_name} <{sender_email}>` in <PageLink id="SiteConfig"/>,
    instead of only `<{sender_email}>`.
  - Add missing filter by `is_exclude_no_salary` to Job Alerts sending.

</ChangelogEntry>
